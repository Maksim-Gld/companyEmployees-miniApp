# App - "companyEmployers-miniApp"

Stack: React / Java Script / HTML / CSS (SASS) / Webpack

\
Web application with various modules:
- Search employer;
- Tab-filter;
- Add new employer;
- Interaction with the selected employee: salary increase/editing/remove.
